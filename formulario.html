Formulários em HTML

Para criar um formulário em HTML, você deve usar a tag "form" para inserir as informações e definir onde começa e onde termina o formulário

Campo de Input

Input: Para fazer um campo digitável no seu formulário, você deve usar a tag "input" e inserir de qual tipo ela é com o "type"

Campo Input com legenda e marca ao fundo

Você também pode adicionar legendas e marcas d'água aos campos de input usando a tag "label" e "placeholder" dentro da tag input. Para a legenda com label, você deve estabelecer um "id" no seu input e um "for" no seu label com o mesmo nome, isso indica para qual input a legenda está relacionada.

Mini formulário para envio

Além to dipo texto, você também pode usar os tipos "Email" e "Password" para criar um formulário. O campo de email só aceita se houver um endereço com "@" e o tipo senha esconde o que está sendo digitado.
Vamos também adicionar um botão para envio do formulário, use o tipo "submit" para isso e o value para dar nome ao botão.

Campos de seleção no formulário em HTML

Você pode adicionar campos de seleção usando as tags do tipo "radio" e "checkbox". O campo radio permite que você selecione apenas uma opção, enquanto o campo checkbox permite que você selecione várias opções.

Observe o exemplo abaixo com seleção usando "radio". Estabeleça um id, um nome, um valor e um label para cada opção. O id define aquele determinado input. O "name" define de qual parte as seleções são. Para que seja selecionado somente uma entre elas. E o "Value" define o valor daquela opção.
