O que são TAGS?

Tags são elementos que definem o conteúdo de uma página, como títulos, parágrafos, imagens, links, etc. As tags são usadas para criar elementos HTML e são definidas por < e >. Algumas são responsáveis por formatar o texto

H1 - H6 Títulos

Temos que os tipos h1 até h6 são os tamanhos de títulos, sendo o h1 o maior e o h6 o menor

Não utilize o formato de titulos para destacar palavras ou frases, isso interfere no formato que o navegador vai interpretar o seu site. Para isso, utilize outra formatação, como negrito, itálico, sublinhado, etc.

Tabelas em HTML

Tabelas: Para criar tabelas em HTML, você deve usar as seguintes tags:
"table" para criar a tabela
"tr" para criar as linhas
"th" para criar o cabeçalho (titulo da coluna), essa tag somente na primeira linha
"td" para criar as células (dados)
E você ainda pode usar alguns estilos, como por exemplo, tamanho das colunas, bordas e alinhamento.

Listas em HTML

Listas não ordenadas

Primeiro vamos fazer uma lista não ordenadas. Faremos isso usando a tag "ul" e por dentro dela a tag "li" que é o item da lista.
Observe o exemplo abaixo de uma lista de cursos da UFMG de computação:

  • Sistemas de Informação
  • Ciência da Computação
  • Engenharia de Sistemas
  • Matemática computacional
  • Ciência de Dados
Observe que a lista ficou marcada com pontinhos, ou seja, é uma lista mas não tem uma ordem específica.

Listas ordenadas

Lista ordenada: Agora vamos fazer uma lista ordenada. Faremos isso usando a tag "ol" e por dentro dela a tag "li" que é o item da lista.
Observe o exemplo abaixo de uma lista de cursos da UFMG de computação:

  1. Sistemas de Informação
  2. Ciência da Computação
  3. Engenharia de Sistemas
  4. Matemática computacional
  5. Ciência de Dados
Observe que agora a lista está ordenada com números, ou seja, ela contém uma ordem, onde o próprio sistema já define a sequencia a partir da quantidade de itens na lista.

Div E HTML Semântico

Div: A tag "div" é uma tag genérica que serve para dividir o conteúdo da página em seções. Ela é muito usada para estilizar o conteúdo da página. Na div, você define uma classe e usa CSS para definir as sessões da página.
Se não quiser usar a Div, você pode usar as principais tags semânticas de HTML para definir o que é cada sessão no seu site.
